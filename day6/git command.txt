Git Branch 

there may be several features in a project divided among the developers. Such features are added bu the developers int their respective branches

1. git branch 
    It shows all the branches and highlights the current branch.

2. git checkout branchname
    It leaves the current branch and jumps to the branch name provided

3. git checkout -b <branchname>
    It creates a new branch and jumps to that branch.
